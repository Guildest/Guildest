# Getting Started with Guildest

Welcome to **Guildest** - the all-in-one Discord server management and analytics bot that helps you understand and grow your community.

## What is Guildest?

Guildest is a powerful Discord bot that provides:

- 📊 **Analytics** - Track message activity, user engagement, and server growth
- 💭 **Sentiment Analysis** - Understand your community's mood with AI-powered insights
- 🛡️ **Moderation Tools** - Keep your server safe with automated moderation
- 📈 **Web Dashboard** - Manage everything from a beautiful web interface
- 🤖 **Slash Commands** - Easy-to-use Discord commands for instant insights

## Quick Start

### 1. Invite the Bot

[Click here to invite Guildest to your server](https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=8&scope=bot%20applications.commands)

Make sure the bot has these permissions:
- Read Messages/View Channels
- Send Messages
- Embed Links
- Read Message History
- Use Slash Commands

### 2. Set Up Your Server

Once the bot joins, the bot will automatically start tracking your server activity. You can customize settings via the web dashboard.

### 3. Access the Dashboard

Visit the dashboard and log in with Discord to access your server's management panel.

From the dashboard, you can:
- View detailed analytics
- Configure server settings
- Generate sentiment reports (Plus/Premium)
- Review moderation logs (Plus/Premium)

## Features Overview

### Analytics

Track your server's activity with detailed metrics:

- **Message Counts** - Hourly message statistics
- **Activity Tracking** - Monitor server engagement
- **Retention History** - 7-90 days depending on your plan

### Sentiment Analysis

Understand your community's emotional tone:

- **Daily Sentiment Scores** - Track positive, neutral, or negative trends
- **AI-Powered Reports** - Detailed insights about community mood (Plus/Premium)
- **Trend Tracking** - See how sentiment changes over time

### Moderation

Keep your server safe and welcoming:

- **Auto-Moderation** - Automatically handle problematic messages
- **Moderation Logs** - Track all moderation actions (Plus/Premium)
- **Configurable Rules** - Customize moderation behavior

### Slash Commands

Quick access to insights directly in Discord:

- `/stats` - View server statistics
- `/sentiment` - Check current sentiment (Plus/Premium)
- `/modlogs` - Review recent moderation actions (Plus/Premium)

## Pricing

Guildest offers three pricing tiers:

- **Free** - Perfect for trying out Guildest (1 server, 7 days analytics)
- **Plus ($9/month)** - Small communities with AI features (3 servers, 30 days, 50 AI reports/month)
- **Premium ($25/month)** - Active communities with full access (10 servers, 90 days, 500 AI reports/month)
